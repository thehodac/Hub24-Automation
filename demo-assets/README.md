# demo-assets/

Screenshots for the Friday demo run sheet (`docs/friday-demo-runsheet.md`).
Do a quick dry run, capture each screenshot, and save it here with the exact
filename below — the run sheet already links to these names.

| File | What to capture |
| --- | --- |
| `02-testcase-page.png` | Confluence "Test Case | Help (Excel+Json)" page (table + JSON). |
| `02-local-testcases.png` | VS Code showing `test-cases/app-header/` (.feature + .json). |
| `03-console.png` | Terminal after a failing run (📊 Total/Passed/Failed + ❌ line). |
| `03-bug-report-docx.png` | The bug report `.docx` open (reason / steps / screenshot). |
| `04-jira-ticket.png` | The auto-created Jira sub-task `[Bug] ...` with attachments. |
| `05-visual-report-docx.png` | The chromatic `.docx` with Before/After/Diff + "Chromatic build:" line. |
| `05-chromatic-build.png` | The Chromatic build page (before/after review). |

Tip: on Windows use **Win + Shift + S** to snip, then paste/save with the name above.
