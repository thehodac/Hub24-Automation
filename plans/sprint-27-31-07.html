4. Per-task approach (Didier's 5 phases each)

Row 38 — Azure DevOps: enable auto-create work items

  1. Research: confirm the Azure reporter is wired (reporters/azuredevops.ts); identify the .env vars (AZURE_DEVOPS_ORG, AZURE_DEVOPS_PROJECT, AZURE_DEVOPS_PAT, AZURE_DEVOPS_CREATE_TICKETS) and how to get the PAT.
  2. Diagram: add Azure DevOps as a bug destination on the auto-ticket sub-process diagram.
  3. Test: fill .env → run node scripts/check-ticket-env.mjs → run a failing test → a Bug work item appears on the right Azure Boards project.
  4. Document: capture evidence (work item created, dedup on re-run, state → Closed on pass).
  5. Secure the flow: verify content (title prefix, Environment/Device, params without password, clear error); mark row 38 Done.

Done when: failing test → Azure work item created · re-run → no duplicate · test passes → work item Closed · content verified.

Row 39 — Publish test cases to Azure DevOps

  1. Research: confirm the Azure target (org/project + link from the user) and the 2 files (Excel .xlsx + JSON).
  2. Diagram: update the "Publishing test cases" step to include Azure DevOps as a destination.
  3. Test: publish an approved sample set to Azure (ASK for destination + link first — side-effectful) → verify the item + both files appear.
  4. Document: save evidence; the repo copy stays source of truth.
  5. Secure the flow: confirm the confirmation gate (never auto-publish without destination + link); mark Done.

Done when: approved test cases published to the agreed Azure location with both files, after explicit user confirmation.

Row 40 — Demo & validation guideline (Azure)

  1. Research: list what to show (failing test → Azure work item → publish test cases) + the audience.
  2. Diagram: reuse the sub-process diagrams as the storyboard.
  3. Build: short guideline doc + a video walkthrough of the Azure flow.
  4. Test: dry-run the guideline so someone else can follow it.
  5. Secure the flow: capture sign-off; mark Done.

Done when: Azure guideline doc + recording exist, dry-run passes, sign-off captured.
